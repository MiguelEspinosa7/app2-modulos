Perfil de Egreso

Información sobre el perfil de Egreso de la Especialidad


La información que ofrece la carrera de Técnico en progamación para permitir el egresado, a través de la articulación de saberes de diversos campos, realizar actividades dirigidas a la:

  • Instalación y desarrollo de sofware de aplicación, utilizando programación estructurada y orientada a objetos en ambientes web y móviles con almacenamiento persistente de datos
  • Así como la configuración y administración de plataformas e-learning y
  • Comercio Electrónico
Durante el proceso de formación de los cincos módulos, el estudiante desarrollará o reforzará:
Las siguientes competencias profesionales:
  • Desarrolla e instala sofware de aplicación utilizando programación estructurtada con almacenamiento persiente de datos
  • Desarrolla sofware de aplicación utilizando programación orientada a objetos, con almacenamieento persistente de datos
  • Desarrolo de aplicaciones web y móviles
  • Administra sistemas operativos, aplicaciones y servicios
  • Desarrolla,administra y configura soluciones de e-learning y comercio electrónico
